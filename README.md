# Foodie Express - Online Food Ordering Website

## Overview
Foodie Express is a simple and interactive online food ordering website developed using HTML, CSS, and JavaScript. The project allows users to browse menu items, add food items to their cart, view the total order amount, and place orders dynamically.

The website demonstrates the implementation of frontend web development concepts, including responsive layouts, DOM manipulation, event handling, and dynamic cart management.

---

## Features
- Interactive online food ordering interface
- Add food items to cart dynamically
- Real-time total amount calculation
- Order summary display
- Responsive and user-friendly design
- Dynamic order confirmation popup
- Automatic footer year update using JavaScript

### 3. script.js

- Total price calculation
- Order placement
- Dynamic DOM updates

---

## Functionalities

### Add to Cart
Users can add food items to their order using interactive buttons.

### Order Summary
Selected items are displayed dynamically in the order section.

### Total Calculation
The total amount updates automatically whenever an item is added.

### Place Order
Displays a summary popup containing:
- Ordered items
- Individual prices
- Total bill amount
